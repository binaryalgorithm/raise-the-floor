# Index

## What This Repository Is

A structured argument that raising human material floors and aligning AI are the same problem — removing interference that prevents coherent systems from operating at their native capacity. The framework is empirical, not ideological. The claims are testable. The evidence is cited.

## Seed Documents

- [**Raise the Floor, Remove the Ceiling**](README.md) — The core framework. Suffering is architectural, not inevitable. Raise the material floor, preserve the ceiling, and human biology does the rest.
- [**Wisdom as Dimensional Coherence**](wisdom_as_coherence.md) — The mechanism. Alignment (human and AI) is a coherence problem, not a constraint problem. Dimensional collapse is the shared failure mode.
